# Secure Enterprise Network Design with Dynamic Routing & Segmentation

## 📌 Overview
This project demonstrates the design and implementation of a secure enterprise network using Cisco Packet Tracer.

The network simulates a real-world enterprise environment with routing, switching, segmentation, and security mechanisms.

---

## 🧠 Network Features

- Dynamic Routing using OSPF
- VLAN segmentation for departments
- Inter-VLAN Routing using a Multilayer Switch
- DHCP for automatic IP address assignment
- Access Control Lists (ACL) for traffic filtering
- ASA Firewall for NAT and perimeter security
- Internet connectivity simulation using Cloud

---

## 🏗️ Network Design

The network consists of:

- 2 Routers (Cisco 2911)
- 1 Multilayer Switch (3560)
- 2 Access Switches (2960)
- ASA Firewall (5506-X)
- End devices (PCs & Laptops)
- Internet Cloud

---

## 🔐 Security Implementation

- ACL rules to control inter-department communication
- NAT configuration on ASA for internet access
- Network segmentation using VLANs

---

## 🌐 Routing

- OSPF is configured for dynamic routing between routers and multilayer switch
- Ensures automatic path selection and failover capability
